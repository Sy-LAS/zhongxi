## 📊 算法复杂度对比

| 算法 | 时间复杂度 | 空间复杂度 | 实时性 |
|------|-----------|-----------|--------|
| ICP | O(N²) | O(N) | ⭐⭐⭐ |
| 特征匹配 | O(F²) | O(F) | ⭐⭐⭐⭐ |
| 差速驱动模型 | O(1) | O(1) | ⭐⭐⭐⭐⭐ |
| 图优化 | O(V³) | O(V+E) | ⭐⭐ |
| 射线追踪 | O(N×M) | O(1) | ⭐⭐⭐⭐ |
| 特征地图管理 | O(F×N) | O(F) | ⭐⭐⭐ |
| 运动补偿 | O(N) | O(N) | ⭐⭐⭐⭐ |
| 前沿探索 | O(W×H) | O(F) | ⭐⭐⭐ |
| 最短路径覆盖 | O(N) | O(N) | ⭐⭐⭐⭐⭐ |
| 回环检测 | O(N) | O(1) | ⭐⭐⭐⭐ |
| 轨迹平滑 | O(N×W) | O(N) | ⭐⭐⭐⭐ |

**符号说明**:
- N:激光点数
- F:特征点数
- V:位姿节点数
- E:约束边数
- W×H:地图尺寸
- W:窗口大小
